Weather Appp

This project aims to create a convenient and intuitive weather application. With it you can get real-time weather information for locations worldwide. The application allows users to search for weather data by city name, ZIP code, or coordinates. Users can save their favorite locations for quick access and enjoy a appealing, responsive interface with dark/light theme options. The platform features elegant animations, pagination for managing multiple saved locations, and persistent storage that remembers user preferences between sessions.

Installation Instructions:
Clone the repository or download the project files
Ensure you have Node.js installed on your system
Install dependencies
Open the project folder in your terminal
Run npm install to install all required packages
Configure API key
Get a free API key from OpenWeatherMap
Replace the placeholder API key in search.ts with your actual ke

Run the application:
Execute npm run dev to start the development server
The application will open in your default browser
